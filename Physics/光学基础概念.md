光学本质上就是 Maxwell equations 在不同尺度下的近似。

## 一、Ray Transfer Matrix Analysis（ABCD Matrix）

### 1.1 第一性原理：为什么可以用"光线"描述光？

从 Maxwell equations 出发，在波长 λ → 0 的极限下，电磁波可以用**几何光学**（geometrical optics）近似。这就是 **eikonal approximation**，类似于量子力学中 ℏ → 0 的经典极限。

**核心思想**：当光学元件尺寸远大于波长，且我们只关心光的传播方向而非波动性时，用"光线"（ray）描述足够精确。

### 1.2 Paraxial Approximation（近轴近似）

这是 ABCD matrix 成立的前提。定义：

$$\sin\theta \approx \tan\theta \approx \theta \ll 1 \text{ rad}$$

即光线与光轴夹角很小（通常 < 10°）。

### 1.3 光线的状态向量

一条光线在某平面的状态用 2×1 列向量表示：

$$\mathbf{v} = \begin{pmatrix} y \\ \theta \end{pmatrix}$$

其中：
- $y$：光线到光轴的垂直距离（高度），单位：m
- $\theta$：光线与光轴的夹角，单位：rad（无量纲但习惯称 rad）

**注意符号约定**：θ > 0 表示光线向上倾斜，θ < 0 表示向下。

### 1.4 基本光学元件的 ABCD Matrix

光线通过光学元件后，状态变为：

$$\mathbf{v}_{out} = M \cdot \mathbf{v}_{in}$$

| 元件 | Matrix M | 推导逻辑 |
|------|----------|----------|
| **Free space** (距离 d) | $\begin{pmatrix} 1 & d \\ 0 & 1 \end{pmatrix}$ | $y_{out} = y_{in} + d \cdot \theta_{in}$, $\theta$ 不变 |
| **Thin lens** (焦距 f) | $\begin{pmatrix} 1 & 0 \\ -1/f & 1 \end{pmatrix}$ | $y$ 不变，$\theta_{out} = \theta_{in} - y/f$ |
| **Flat interface** (n₁→n₂) | $\begin{pmatrix} 1 & 0 \\ 0 & n_1/n_2 \end{pmatrix}$ | Snell's law 近轴形式 |
| **Curved interface** (半径 R) | $\begin{pmatrix} 1 & 0 \\ \frac{n_1-n_2}{R n_2} & \frac{n_1}{n_2} \end{pmatrix}$ | 结合 Snell + 曲面几何 |
| **Mirror** (曲率半径 R) | $\begin{pmatrix} 1 & 0 \\ -2/R & 1 \end{pmatrix}$ | 等效于 f = R/2 的 lens |

### 1.5 为什么是 2×2 Matrix？——Phase Space 的视角

这与 Hamiltonian mechanics 中的 **phase space** 完全类比：

| 经典力学 | 几何光学 |
|----------|----------|
| Position $q$ | Height $y$ |
| Momentum $p$ | Angle $\theta$ (或光学方向余弦) |
| Hamilton's equations | Ray equation |
| Canonical transformation | ABCD matrix |
| Liouville's theorem | Étendue conservation |

**关键洞察**：ABCD matrix 就是光线在 **phase space** 中的 canonical transformation！

### 1.6 复杂光学系统的计算

多个元件串联时，matrix 乘法从输出端向输入端**反向**叠加：

$$M_{total} = M_N \cdot M_{N-1} \cdot ... \cdot M_2 \cdot M_1$$

**Example**：两 lens 系统的 matrix

```
[Input] --d1--> [Lens f1] --d2--> [Lens f2] --d3--> [Output]

M_total = M_space(d3) × M_lens(f2) × M_space(d2) × M_lens(f1) × M_space(d1)
```

### 1.7 实际应用：成像条件

当 $M_{12} = 0$（即 matrix 元素 (1,2) = 0），输入平面与输出平面满足**成像关系**：

$$y_{out} = M_{11} \cdot y_{in}$$

此时 $M_{11}$ 就是**放大倍率**（magnification）。

## 二、Optical Cavity（光腔）

### 2.1 什么是 Optical Cavity？

**Optical cavity**（也称为 optical resonator）是由两个或多个 mirrors 构成的结构，使光在其中来回反射，形成**共振模式**。

最基本的结构：**Fabry-Pérot cavity**（法布里-珀罗腔）

```
        Mirror M1          Mirror M2
          |                    |
    R1 <──|────────────────────|──> R2
          |    Length L        |
          |                    |
```

### 2.2 Stability Condition（稳定性条件）

光线在腔内来回反射 n 次后的状态：

$$\mathbf{v}_n = M_{round-trip}^n \cdot \mathbf{v}_0$$

**稳定腔**要求光线不会发散，即 $|\text{eigenvalues}| = 1$。

Round-trip matrix（从 M1 出发，到 M2 再返回 M1）：

$$M_{rt} = \begin{pmatrix} 1 & L \\ 0 & 1 \end{pmatrix} \begin{pmatrix} 1 & 0 \\ -2/R_2 & 1 \end{pmatrix} \begin{pmatrix} 1 & L \\ 0 & 1 \end{pmatrix} \begin{pmatrix} 1 & 0 \\ -2/R_1 & 1 \end{pmatrix}$$

简化后 trace：

$$\text{Tr}(M_{rt}) = 2 - 4L\left(\frac{1}{R_1} + \frac{1}{R_2} - \frac{L}{R_1 R_2}\right)$$

定义 **g-parameters**：

$$g_1 = 1 - \frac{L}{R_1}, \quad g_2 = 1 - \frac{L}{R_2}$$

**Stability condition**（稳定条件）：

$$\boxed{0 \leq g_1 \cdot g_2 \leq 1}$$

### 2.3 稳定性图解

```
    g2
     │
   1 ┼───────┬───────┐
     │ STABLE│STABLE │
     │       │       │
   0 ┼───────┼───────┤  Unstable
     │       │       │
  -1 ┼───────┴───────┘
     └───────┼───────┼────── g1
            -1      1
```

### 2.4 Cavity Modes（腔模）

稳定腔内存在离散的 **Gaussian modes**：

$$TEM_{mn}$$ 模式的共振频率：

$$\nu_{mnq} = \frac{c}{2L}\left[q + \frac{m+n+1}{\pi}\arccos\sqrt{g_1 g_2}\right]$$

其中：
- $q$：axial mode number（纵模数），通常 ~10⁵ - 10⁶
- $m, n$：transverse mode numbers（横模数），0, 1, 2, ...
- $L$：cavity length

### 2.5 应用

| 应用 | Cavity 类型 | 特点 |
|------|-------------|------|
| **Laser** | Stable cavity | 形成高 gain 模式 |
| **Interferometer** | Fabry-Pérot | 高分辨率光谱 |
| **LIGO** | Power recycling cavity | 增强有效光程 |
| **Optical clock** | Ultra-stable cavity | Q > 10¹⁰ |

**Reference**: 
- https://en.wikipedia.org/wiki/Optical_cavity
- https://www.rp-photonics.com/optical_resonators.html

## 三、Symplectic Matrix（辛矩阵）

### 3.1 定义

$2n \times 2n$ 实矩阵 $M$ 称为 **symplectic**，若满足：

$$M^T J M = J$$

其中 $J$ 是 **symplectic form**：

$$J = \begin{pmatrix} 0 & I_n \\ -I_n & 0 \end{pmatrix}$$

对于 $n=1$（光学系统的 case）：

$$J = \begin{pmatrix} 0 & 1 \\ -1 & 0 \end{pmatrix}$$

### 3.2 为什么 ABCD Matrix 是 Symplectic？

**在 uniform refractive index 介质中**，ABCD matrix 满足：

$$\det(M) = 1$$

这正是 $n=1$ 时 symplectic condition 的简化形式！

更一般地，当折射率从 $n_1$ 变到 $n_2$：

$$\det(M) = \frac{n_1}{n_2}$$

这可以理解为 **modified symplectic structure**。

### 3.3 物理意义：Liouville's Theorem

**Symplectic matrix 保持 phase space 体积不变！**

在光学中，这对应 **étendue conservation**（光展守恒）：

$$\mathcal{E} = n^2 \cdot A \cdot \Omega = \text{const}$$

其中：
- $n$：refractive index
- $A$：area
- $\Omega$：solid angle

**直觉**：你不能用 passive optical elements 把光"压缩"到任意小的 phase space volume——这违反热力学第二定律！

### 3.4 与 Hamiltonian Mechanics 的联系

Hamilton's equations：

$$\dot{q}_i = \frac{\partial H}{\partial p_i}, \quad \dot{p}_i = -\frac{\partial H}{\partial q_i}$$

时间演化矩阵 $M(t)$ 必定是 symplectic，因为：

$$\frac{d}{dt}(M^T J M) = 0$$

**光学类比**：光线传播（z 方向作为"时间"）完全类似 Hamiltonian flow！

| 力学 | 光学 |
|------|------|
| Time $t$ | Axial coordinate $z$ |
| Position $q$ | Height $y$ |
| Momentum $p$ | Optical direction cosine $n\sin\theta \approx n\theta$ |
| Hamiltonian $H(q,p)$ | Ray Hamiltonian $\mathcal{H}(y, n\theta)$ |

### 3.5 Symplectic Eigenvalues

Symplectic matrix 的 eigenvalues 成对出现：若 $\lambda$ 是 eigenvalue，则 $1/\lambda$ 和 $\bar{\lambda}$ 也是。

对于稳定光学系统，所有 eigenvalues 在 unit circle 上：$|\lambda| = 1$。

**Reference**:
- https://en.wikipedia.org/wiki/Symplectic_matrix
- https://en.wikipedia.org/wiki/Symplectic_vector_space

## 四、Fermat's Principle（费马原理）

### 4.1 表述

**Fermat's principle**（1657 年提出，1662 年完善）：

> 光在两点之间传播的路径是**光程取极值**的路径。

$$\delta \int_A^B n(\mathbf{r}) \, ds = 0$$

其中：
- $n(\mathbf{r})$：位置 $\mathbf{r}$ 处的 refractive index
- $ds$：路径微元
- 光程 = $\int n \, ds$

### 4.2 与 Principle of Least Action 的对应

这完全类比于力学中的 **Maupertuis principle**（最小作用量原理）：

| 力学 | 光学 |
|------|------|
| Action $\int p \, dq$ | Optical path $\int n \, ds$ |
| Lagrangian $L = T - V$ | Optical Lagrangian $\mathcal{L} = n\sqrt{1 + y'^2 + x'^2}$ |
| Hamilton's principle | Fermat's principle |

### 4.3 从 Fermat 推导 Snell's Law

界面两侧 $n_1, n_2$，光程：

$$S = n_1 \sqrt{x^2 + h_1^2} + n_2 \sqrt{(d-x)^2 + h_2^2}$$

极值条件 $\partial S / \partial x = 0$：

$$\frac{n_1 x}{\sqrt{x^2 + h_1^2}} - \frac{n_2 (d-x)}{\sqrt{(d-x)^2 + h_2^2}} = 0$$

即：

$$n_1 \sin\theta_1 = n_2 \sin\theta_2$$

这就是 **Snell's law**！

### 4.4 变分形式与 Euler-Lagrange 方程

设路径 $y(x)$，光程：

$$S[y] = \int_{x_A}^{x_B} n(x,y) \sqrt{1 + \left(\frac{dy}{dx}\right)^2} \, dx$$

Euler-Lagrange 方程：

$$\frac{d}{dx}\left(\frac{\partial \mathcal{L}}{\partial y'}\right) - \frac{\partial \mathcal{L}}{\partial y} = 0$$

其中 $\mathcal{L} = n\sqrt{1 + y'^2}$。

这给出 **ray equation**：

$$\frac{d}{ds}\left(n\frac{d\mathbf{r}}{ds}\right) = \nabla n$$

### 4.5 极值不一定是极小值！

Fermat 最初说的是"最短时间"，但实际是"极值"（stationary）。例子：

- **Ellipsoidal mirror**：从一个焦点到另一个焦点的所有路径光程相等（不是极小！）
- **Saddle point 情况**：某些光学系统中光程既非极小也非极大

**Reference**:
- https://en.wikipedia.org/wiki/Fermat%27s_principle
- https://feynmanlectures.caltech.edu/I_26.html

## 五、Eikonal Equation（程函方程）

### 5.1 从 Wave Equation 推导

从 Maxwell equations 得到的 scalar wave equation（标量波动方程）：

$$\nabla^2 E - \frac{n^2}{c^2}\frac{\partial^2 E}{\partial t^2} = 0$$

设解的形式为：

$$E(\mathbf{r}, t) = A(\mathbf{r}) e^{i k_0 S(\mathbf{r}) - i\omega t}$$

其中：
- $k_0 = \omega/c = 2\pi/\lambda_0$：vacuum wave number
- $S(\mathbf{r})$：**eikonal function**（程函），物理意义是 optical path length
- $A(\mathbf{r})$：amplitude

代入并取 $\lambda \to 0$（即 $k_0 \to \infty$）极限，最高阶项给出：

$$\boxed{|\nabla S|^2 = n^2(\mathbf{r})}$$

这就是 **eikonal equation**。

### 5.2 物理意义

- $S(\mathbf{r}) = \text{const}$ 定义了 **wavefronts**（等相位面）
- $\nabla S$ 指向光线传播方向
- $|\nabla S| = n$ 说明光在介质中的"光学速度"为 $c/n$

### 5.3 与 Hamilton-Jacobi 方程的联系

Eikonal equation 与 **Hamilton-Jacobi equation** 形式完全相同：

$$\left(\frac{\partial S}{\partial x}\right)^2 + \left(\frac{\partial S}{\partial y}\right)^2 + \left(\frac{\partial S}{\partial z}\right)^2 = n^2$$

对应 HJ 方程：

$$\left(\frac{\partial S}{\partial q_i}\right)^2 = 2m(E - V)$$

**深层联系**：
- Geometrical optics ↔ Classical mechanics
- Wave optics ↔ Quantum mechanics
- $\hbar \to 0$ 对应 $\lambda \to 0$

### 5.4 从 Eikonal 推导 Ray Equation

定义光线方向向量：

$$\mathbf{s} = \frac{\nabla S}{|\nabla S|} = \frac{\nabla S}{n}$$

可以证明：

$$\frac{d}{ds}\left(n\frac{d\mathbf{r}}{ds}\right) = \nabla n$$

这就是前面 Fermat principle 给出的 ray equation！

### 5.5 应用

| 领域 | 应用 |
|------|------|
| **Lens design** | 追迹非均匀介质中的光线 |
| **Atmospheric optics** | 计算大气折射 |
| **Gradient index optics** | GRIN lens 设计 |
| **Seismic wave propagation** | 地震波射线追踪 |

**Reference**:
- https://en.wikipedia.org/wiki/Eikonal_equation
- https://en.wikipedia.org/wiki/Geometrical_optics

## 七、WKB Approximation 回顾

### 7.1 什么是 WKB？

**WKB approximation**（Wentzel-Kramers-Brillouin, 1926）是求解一维 Schrödinger 方程的半经典近似方法。

核心思想：当 de Broglie wavelength $\lambda = h/p$ 远小于势场变化的特征长度时，波函数近似为：

$$\psi(x) \approx A(x) e^{\pm i \int^x k(x') dx' / \hbar}$$

### 7.2 推导

一维 Schrödinger equation：

$$-\frac{\hbar^2}{2m}\frac{d^2\psi}{dx^2} + V(x)\psi = E\psi$$

设解为：

$$\psi(x) = e^{i S(x)/\hbar}$$

代入得：

$$\left(\frac{dS}{dx}\right)^2 - i\hbar\frac{d^2S}{dx^2} = 2m(E - V) \equiv p^2(x)$$

**关键近似**：$\hbar \to 0$（或等价地，$p(x)$ 变化缓慢）

展开 $S = S_0 + \hbar S_1 + ...$：

- $O(\hbar^0)$: $\left(\frac{dS_0}{dx}\right)^2 = p^2(x) \Rightarrow S_0 = \pm \int p(x) dx$
- $O(\hbar^1)$: $\frac{dS_1}{dx} = -\frac{1}{2}\frac{p'}{p} \Rightarrow S_1 = -\frac{1}{2}\ln p$

最终解：

$$\psi_{WKB}(x) \approx \frac{C_1}{\sqrt{p(x)}} e^{i\int^x p(x')dx'/\hbar} + \frac{C_2}{\sqrt{p(x)}} e^{-i\int^x p(x')dx'/\hbar}$$

### 7.3 WKB 与几何光学的深刻联系

**这就是量子力学中的 eikonal approximation！**

| 量子力学 WKB | 几何光学 |
|--------------|----------|
| $\hbar$ | $\lambda$ |
| $p(x) = \sqrt{2m(E-V)}$ | $n(\mathbf{r})$ |
| $S(x) = \int p\,dx$ | $S(\mathbf{r}) = \int n\,ds$ |
| HJ equation | Eikonal equation |
| Classical limit $\hbar \to 0$ | Geometric limit $\lambda \to 0$ |

**统一框架**：两者都是 wave equation 在短波长极限下的近似！

### 7.4 经典转折点

$E = V(x)$ 处 $p(x) = 0$，WKB 发散。需要用 **connection formulas** 连接振荡区和衰减区。

**Airy function matching**：

在转折点附近，用线性势近似 $V(x) \approx V(x_0) + V'(x_0)(x-x_0)$，精确解为 Airy function：

$$\psi(x) \propto \text{Ai}\left[\left(\frac{2mV'}{\hbar^2}\right)^{1/3}(x - x_0)\right]$$

### 7.5 应用

| 应用 | 具体例子 |
|------|----------|
| **Tunneling** | Alpha decay, Zener tunneling |
| **Bound states** | Bohr-Sommerfeld quantization |
| **Scattering** | WKB phase shifts |
| **Semiclassical limit** | Quantum chaos |

**Bohr-Sommerfeld 条件**：

$$\oint p(x) dx = \left(n + \frac{1}{2}\right)h, \quad n = 0, 1, 2, ...$$

这给出了谐振子能级的正确结果！

**Reference**:
- https://en.wikipedia.org/wiki/WKB_approximation
- Griffiths QM, Chapter 8

## 八、Refractive Index 在空间上变化？

### 8.1 你的困惑是合理的！

对于**均匀材料**（homogeneous material），如一块纯石英玻璃，RI 确实在 3D 空间上是常数。标准教科书的表格给出：

| 材料 | Refractive Index (n) |
|------|---------------------|
| Vacuum | 1.000000 |
| Air (STP) | 1.000293 |
| Water | 1.333 |
| Glass (crown) | ~1.52 |
| Diamond | 2.42 |

**但实际情况更复杂！**

### 8.2 RI 变化的物理来源

#### (1) **大气/气体：温度和压力梯度**

大气的 $n$ 随温度 $T$ 和压力 $P$ 变化：

$$n_{air} - 1 \propto \frac{P}{T}$$

在毫米到米尺度上，温度梯度导致：

- **Mirage**（海市蜃楼）
- **Atmospheric turbulence**（大气湍流）
- **Astronomical seeing**（天文视宁度）

**典型数值**：温度变化 1 K 导致 $\Delta n \sim 10^{-6}$，听起来很小，但在长路径上累计效应显著！

#### (2) **固体材料：应力/温度梯度**

- **Photoelastic effect**：应力改变材料的 RI（用于应力分析）
- **Thermal lensing**：激光加热材料，产生 RI 梯度
- **Photorefractive effect**：光照改变某些晶体的 RI

#### (3) **人为设计的梯度材料：GRIN Optics**

**Gradient Index (GRIN)** 材料是故意制造的 RI 非均匀材料！

**应用**：
- **GRIN lens**：flat surfaces 但有聚焦功能（手机摄像头中广泛使用）
- **GRIN fiber**：光纤中 RI 从中心向边缘递减
- **Eye lens**：人眼晶状体本身就是 GRIN！

### 8.3 GRIN Lens 的数学描述

最常见的是 **parabolic index profile**：

$$n(r) = n_0 \left(1 - \frac{1}{2} \alpha r^2\right)$$

其中：
- $n_0$：轴上 RI
- $\alpha$：gradient parameter，单位 m⁻²
- $r$：到光轴距离

**光线方程**变为：

$$\frac{d^2r}{dz^2} + \alpha r = 0$$

解为 **sinusoidal path**：

$$r(z) = r_0 \cos(\sqrt{\alpha} z) + \frac{\theta_0}{\sqrt{\alpha}}\sin(\sqrt{\alpha} z)$$

光线在 GRIN 介质中**自动聚焦**，周期为 $2\pi/\sqrt{\alpha}$！

### 8.4 实际例子：光纤

**Step-index fiber** vs **GRIN fiber**：

```
Step-index:          GRIN:
  n1 │███│           n(r) ∝ 1 - r²
     │███│               ╭──────╮
  n2 │   │           n0 │   ⬤  │
     └───┘               ╰──────╯
```

GRIN fiber 的优点：
- 更大的 bandwidth
- 减少模间色散

### 8.5 为什么"毫米尺度"重要？

你提到"毫米尺度变化"，这暗示：

1. **几何光学仍适用**：如果 RI 变化尺度 >> λ，仍可用 ray optics
2. **但是 ABCD matrix 不够**：需要解 ray equation 而非用 constant matrices

**长度尺度比较**：
- 可见光波长：~0.5 μm
- RI 变化尺度：~mm = 1000 μm
- 比值：~2000 >> 1

所以 geometrical optics 仍然有效，但需要 **continuous ray tracing**。

### 8.6 数学描述：一般情况

一般介质 $n(\mathbf{r})$ 中的 ray equation：

$$\frac{d}{ds}\left(n\frac{d\mathbf{r}}{ds}\right) = \nabla n$$

展开：

$$n\frac{d^2\mathbf{r}}{ds^2} + \left(\frac{dn}{ds}\right)\frac{d\mathbf{r}}{ds} = \nabla n$$

对于 paraxial rays（$ds \approx dz$），简化为：

$$\frac{d^2\mathbf{r}}{dz^2} = \frac{1}{n}\nabla_\perp n$$

### 8.7 一个具体数值例子

**大气湍流**：

- 海平面 $n \approx 1.000293$
- 温度变化 $\Delta T \sim 1$ K over $\sim 1$ mm
- $\Delta n \sim 10^{-6}$
- 累积效应：100 m 路径上光束偏折角 $\sim 10^{-6}$ rad $\times 10^5 = 0.1$ rad！

这就是为什么星星会"闪烁"。
