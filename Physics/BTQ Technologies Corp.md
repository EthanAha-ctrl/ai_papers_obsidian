## BTQ Technologies Corp. — 简要概述

**BTQ Technologies** (Nasdaq: BTQ / CBOE CA: BTQ / FSE: NG3) 是一家总部位于加拿大 Vancouver 的**全球量子技术公司**，核心聚焦于两个方向：

### 1. Post-Quantum Cryptography (PQC) — 抗量子密码学
量子计算机有望在未来打破现有的公钥加密体系（如 RSA、ECC），BTQ 的核心使命就是在"量子威胁"到来**之前**，提供软硬件一体的安全解决方案，保护关键基础设施与数据。

**主要产品线：**

| 产品 | 说明 |
|------|------|
| **QCIM** (Quantum Cryptographic Integrated Module) | 量子安全芯片，硬件级 PQC 安全根，与 ITRI 合作在硅片上验证 |
| **QSSN** (Quantum Secure Security Network) | 量子安全网络，面向关键通信基础设施 |
| **QPOW** (Quantum Proof of Work) | 量子安全的工作量证明，面向区块链场景 |

### 2. Quantum Computing — 量子计算
通过收购 **QPerfect**（中性原子量子计算公司），BTQ 已转型为**全栈量子公司**（从量子硬件到量子算法到后量子安全）。同时在量子纠错理论方面有前沿研究——首席量子官 Dr. Gavin K. Brennen 联合发表了关于 **Permutation-Invariant Codes** 通用纠错理论的首篇一般性论文。

### 3. 生态与合作
- 与 **Bonsol Labs** 合作在 **Solana** 链上实现了业界首个 NIST 标准 PQC 签名验证
- 战略投资韩国 **Keypair**，获取 PQC IP，推进韩国市场
- 被纳入 **VanEck Quantum Computing UCITS ETF**，欧洲投资人可通过合规 UCITS 渠道投资
- 获得澳大利亚政府支持，在悉尼推进研发运营

---

**一句话总结：** BTQ 是一家做**抗量子密码学（PQC）硬件/软件解决方案 + 量子计算全栈**的公司，目标是在量子计算威胁现实化之前，为区块链、金融、政府等关键领域提供"量子安全"的基础设施。
