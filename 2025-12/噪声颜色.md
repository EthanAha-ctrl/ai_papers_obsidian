## 文章概述

这是 Casey Muratori 写的一篇关于**噪声颜色**的技术文章，发表于他的博客 series "The Witness"。文章从一个实际的 raytracing 渲染问题出发，深入讲解了不同"颜色"的噪声及其在计算机图形学中的应用，特别是**蓝噪声采样模式**的生成算法优化。

## 核心概念解析

### 1. 噪声的"颜色"分类

噪声的颜色分类是基于**功率谱密度**的频率分布特性，类比于可见光谱：

| 噪声颜色 | 频率特性 | 空间分布特征 | 功率谱密度 $S(f)$ |
|---------|---------|-------------|------------------|
| **White Noise（白噪声）** | 所有频率均等 | 纯随机点，有聚集和空隙 | $S(f) \propto f^0$（常数） |
| **Blue Noise（蓝噪声）** | 只有高频，无低频 | 点均匀分布，无聚集 | $S(f) \propto f^1$ |
| **Red/Pink Noise（红噪声）** | 只有低频，无高频 | 点高度聚集 | $S(f) \propto f^{-2}$ |
| **Green Noise（绿噪声）** | 只有中频 | 介于蓝红之间 | $S(f) \propto f^{-1}$ |

> **第一性原理理解**：这里的"颜色"是类比可见光谱。蓝光波长最短、频率最高，所以"蓝噪声"意味着**高频能量占主导**——在空间域上表现为点与点之间保持均匀距离，没有"紧密聚集"（低频成分）。

### 2. 为什么蓝噪声对图像采样重要？

文章通过一个 raytraced Japanese cabinet stairway 的例子说明：

**问题**：使用规则的 4×4 sub-pixel grid 采样时，边缘出现锯齿伪影。

**原因**：
- 规则采样的周期性会产生**moiré patterns**和**aliasing**
- 当边缘穿过多个像素时，每个像素的规则采样会产生同步的颜色变化，形成视觉上明显的"条纹"

**解决方案**：使用不规则的采样模式——引入噪声破坏周期性。

但**不是所有噪声都等价**：
- White noise：随机性太强，会产生明显的"clumps"（聚集）和"voids"（空隙）
- Blue noise：点分布均匀，既打破周期性又不会产生明显的空隙或聚集

这正是为什么人眼的感光细胞排列被观察到类似蓝噪声模式——进化的选择。

## 算法详解：Poisson Disk Sampling

### 基础算法（Naïve Approach）

```
Algorithm 1: Naïve Poisson Disk Sampling
Input: minimum distance r, region bounds
Output: point set P

1. P ← ∅
2. while not done:
3.     candidate ← random_point(entire_region)
4.     if ∀p ∈ P: distance(candidate, p) ≥ r:
5.         P ← P ∪ {candidate}
6. return P
```

**核心约束**：任意两点间距离 ≥ r

**公式表达**：
$$\forall p_i, p_j \in P, i \neq j: \|p_i - p_j\| \geq r$$

### 计算复杂度问题

文章给出了关键洞察——随着空间被填满，算法效率急剧下降：

设已接受的点数为 $n$，覆盖的区域比例为 $\theta$，则：

$$\theta \approx n \cdot \frac{\pi r^2}{A}$$

其中 $A$ 是总区域面积。

**接受率**：
$$P(\text{accept}) \approx 1 - \theta = 1 - \frac{n\pi r^2}{A}$$

**期望采样次数**：
$$E[\text{samples per accepted point}] = \frac{1}{P(\text{accept})} = \frac{1}{1 - \frac{n\pi r^2}{A}}$$

当 $\theta \to 1$ 时，$E \to \infty$！

这就是文章中那张"爆炸式增长"曲线的数学本质：
$$\text{Samples per point} = O\left(\frac{1}{1 - n\pi r^2/A}\right)$$

## 算法改进：Wavefront Propagation Method

### 核心思想

**从全局随机采样 → 局部邻域采样**

不要在整个空间盲目采样，而是**只从已有点的邻域**采样。

### 详细算法

```
Algorithm 2: Fast Poisson Disk Sampling (Wavefront)
Input: minimum distance r, max attempts k
Output: point set P

1. P ← ∅
2. active_list ← ∅
3. initial_point ← random_point(region)
4. P ← {initial_point}
5. active_list ← {initial_point}

6. while active_list ≠ ∅:
7.     randomly select base_point from active_list
8.     for i = 1 to k:
9.         θ ← uniform(0, 2π)        // random angle
10.        d ← uniform(r, 2r)         // random distance in annulus
11.        candidate ← base_point + d·(cos θ, sin θ)
12.        if valid(candidate, P, r):
13.            P ← P ∪ {candidate}
14.            active_list ← active_list ∪ {candidate}
15.            break
16.    if no valid candidate found in k attempts:
17.        active_list ← active_list \ {base_point}

18. return P
```

### 关键公式：Annulus Sampling

新候选点的采样区域是一个**环形区域**：

$$\text{Candidate region: } r \leq d \leq 2r$$

使用极坐标：
$$\begin{cases} x_{\text{candidate}} = x_{\text{base}} + d \cos\theta \\ y_{\text{candidate}} = y_{\text{base}} + d \sin\theta \end{cases}$$

其中：
- $\theta \sim \text{Uniform}(0, 2\pi)$
- $d \sim \text{Uniform}(r, 2r)$

### 为什么选择 [r, 2r]？

**下界 r**：保证最小距离约束
$$d \geq r \Rightarrow \|p_{\text{new}} - p_{\text{base}}\| \geq r$$

**上界 2r**：保证不会留下过大的空洞

如果两点距离 > 2r，那么在它们连线的中点处，可能存在一个直径 > r 的圆形空洞，可以放入一个新点。

所以，如果所有点对之间距离 ≤ 2r，理论上空间已被"紧凑填满"。

### 空间划分加速：Background Grid

实际实现中，需要快速查询"附近是否有点"。常用方法是使用**背景网格**：

- 网格大小：$g = \frac{r}{\sqrt{2}}$（保证每个格子最多一个点）
- 对于候选点 $p$，只需检查相邻 5×5 网格中的点

$$\text{Grid index of point } p = \left(\left\lfloor \frac{x}{g} \right\rfloor, \left\lfloor \frac{y}{g} \right\rfloor\right)$$

**查询复杂度**：从 $O(n)$ 降到 $O(1)$！

## 复杂度分析对比

| 方法 | 时间复杂度 | 空间复杂度 | 拒绝率 |
|-----|----------|-----------|--------|
| Naïve | $O(n \cdot \frac{1}{1-\theta})$ | $O(n)$ | 指数增长 |
| Wavefront | $O(kn)$ | $O(n)$ + Grid $O(\frac{A}{r^2})$ | 常数 |

其中 $k$ 是每点的最大尝试次数（通常 30-64），$\theta$ 是填充密度。

## 在 The Witness 中的应用

### 草地放置系统

游戏需要在大面积地形上放置草/植被，要求：
- 视觉上自然（不能是规则网格）
- 没有明显的聚集或空隙
- 性能可接受

原代码问题：
1. 使用 naïve Poisson disk sampling
2. 在检查有效性**之前**就计算 3D 地形位置（需要 raycast）
3. 随着拒绝率升高，每帧大量 raycast 被浪费

改进后：
- Wavefront 方法使拒绝率保持稳定
- 大幅减少无效的 raycast 计算

## 文章预告的"理论问题"

文章结尾提到还有一个"理论问题"未解决。这通常指的是：

**Poisson Disk Sampling 的理论缺陷**：标准方法生成的分布不满足真正的 Poisson 过程统计特性。真正的 Poisson disk distribution 要求：
$$P(\text{点位于区域 } A) = \lambda |A| \cdot \exp(-\lambda|A|)$$

而常用的 rejection-based 方法只满足"hard-core"条件，不满足完整的 Poisson 统计。

## 数学深度：功率谱密度与噪声颜色

### PSD (Power Spectral Density) 定义

对于一维信号 $x(t)$，其功率谱密度：
$$S_x(f) = \lim_{T \to \infty} \frac{1}{T} \left| \mathcal{F}\{x_T(t)\}(f) \right|^2$$

其中 $\mathcal{F}$ 是傅里叶变换。

对于二维点分布，可以计算其**periodogram**：
$$S(k_x, k_y) = \left| \sum_{p \in P} \exp(-i(k_x x_p + k_y y_p)) \right|^2$$

### Blue Noise 的 PSD 特征

理想的蓝噪声分布：
$$S(f) \propto \begin{cases} 0 & f < f_{\text{min}} \\ \text{const} & f \geq f_{\text{min}} \end{cases}$$

实际中，$S(f)$ 在低频区域接近零，高频区域趋于平坦。

### Noise Color 的统一公式

$$S(f) \propto f^\beta$$

| $\beta$ 值 | 噪声类型 |
|-----------|---------|
| $\beta = 0$ | White noise |
| $\beta = 1$ | Pink noise |
| $\beta = 2$ | Brownian/Red noise |
| $\beta = -1$ | Blue noise |
| $\beta = -2$ | Violet noise |
